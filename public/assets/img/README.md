# Image Assets

Please place the following images in this directory:

1. **dare1.png** - Dare Centre logo (used in navbar)
2. **aicraaa.jpg** - AICRA logo (used in navbar)

These images should be copied from the existing certificate-verification project or from the Dare Centre website assets.

## Image Requirements

- **dare1.png**: Recommended size ~200x60px, PNG format with transparency
- **aicraaa.jpg**: Recommended size ~200x80px, JPG format

If you don't have these images, you can temporarily use placeholder images or download them from the Dare Centre website.
